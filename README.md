# Expenses Tracker

[![Run in Codespaces](https://github.com/codespaces/badge.svg)](https://github.com/codespaces/new?repo=riasanto97-py/expenses-tracker&devcontainer_path=.devcontainer&startup_script=python%20main.py)

Ένα απλό Python πρόγραμμα για παρακολούθηση και διαχείριση προσωπικών εξόδων μέσω της κονσόλας (CLI).  
Το πρόγραμμα είναι **για εκπαιδευτικούς σκοπούς** και αποθηκεύει τα έξοδα σε μια τοπική **SQLite βάση δεδομένων** που δημιουργείται αυτόματα (`expenses.db`).

---

## Οδηγίες για το Codespaces

1. Κάντε κλικ στο κουμπί **Run in Codespaces** παραπάνω.  
2. Ανοίγει ένα νέο Codespace με το project.  
3. Ανοίξτε το **Terminal** μέσα στο Codespace.  
4. Πληκτρολογήστε την εντολή:

```bash
python main.py
