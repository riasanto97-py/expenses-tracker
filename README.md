
# Expenses Tracker

[![Run in Codespaces](https://github.com/codespaces/badge.svg)](https://github.com/codespaces/new?repo=riasanto97-py/expenses-tracker)

Μια απλή εφαρμογή γραμμής εντολών (CLI) γραμμένη σε Python που επιτρέπει στον χρήστη να καταγράφει, να προβάλλει και να διαχειρίζεται τα καθημερινά του έξοδα.
Τα δεδομένα αποθηκεύονται τοπικά σε μια βάση δεδομένων **SQLite**.

Το project δημιουργήθηκε για **εκπαιδευτικούς σκοπούς**, με στόχο την εξάσκηση σε βασικές έννοιες της Python όπως:

* Αντικειμενοστραφής Προγραμματισμός (Object-Oriented Programming)
* Αλληλεπίδραση με βάση δεδομένων SQLite
* Οργάνωση κώδικα σε αρθρωτή δομή (modular structure)
* Έλεγχος εγκυρότητας δεδομένων (input validation)
* Δημιουργία εφαρμογών γραμμής εντολών (CLI)

# Εκτέλεση του Project στο GitHub Codespaces

Μπορείτε να εκτελέσετε την εφαρμογή απευθείας μέσα από τον browser χωρίς να εγκαταστήσετε τίποτα στον υπολογιστή σας.

1. Πατήστε το κουμπί **Run in Codespaces** παραπάνω.

5. Ακολουθήστε τις επιλογές του μενού που εμφανίζονται στο terminal.

# Χαρακτηριστικά

* Προσθήκη νέου εξόδου με:

  * Ποσό
  * Κατηγορία
  * Περιγραφή
  * Ημερομηνία
* Προβολή όλων των εξόδων σε μορφή πίνακα
* Διαγραφή εξόδου βάσει του `ID`
* Αυτόματη δημιουργία της βάσης δεδομένων
* Έλεγχος εγκυρότητας για ποσά και ημερομηνίες
* Απλό και καθαρό περιβάλλον γραμμής εντολών

* Python 3.7 ή νεότερη έκδοση

# Δομή του Project

```text
expenses-tracker
│
├── main.py
├── database.py
├── models.py
├── README.md
└── expenses.db (δημιουργείται αυτόματα όταν εκτελείται το πρόγραμμα)
```

**Περιγραφή αρχείων**

* **main.py**
  Περιέχει το βασικό μενού της εφαρμογής και τη διαχείριση της αλληλεπίδρασης με τον χρήστη.

* **database.py**
  Περιέχει όλες τις λειτουργίες που σχετίζονται με τη βάση δεδομένων SQLite (δημιουργία πίνακα, εισαγωγή, ανάκτηση και διαγραφή δεδομένων).

* **models.py**
  Ορίζει την κλάση `Expense`, η οποία αναπαριστά ένα έξοδο.
